# Mini E-Commerce Frontend

React + Vite UI for the Mini E-Commerce API.

## Requirements

- Node.js 18+
- Backend running on port `8080`

## Setup

```bash
cd mini-ecommerce-frontend
cp .env.example .env
npm install
```

## Run

```bash
cd mini-ecommerce-frontend
npm run dev
```

## Build

```bash
cd mini-ecommerce-frontend
npm run build
```
